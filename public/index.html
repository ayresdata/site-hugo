PostgreSQL

PostgreSQL es una de las principales opciones para Bases de Datos Relacionales que requieran estructuras lógicas complejas. Junto con MySQL y MariaDB, compone el ecosistema de bases relacionales libres. La comunidad de PostgreSQL ha sabido llevar a este motor, hacia un framework de datos.

Redshift

Redshift es una tecnología basada en PostgreSQL y Redsfhit, diseñada para implementar almacenamiento de grandes cantidades de datos de manera compacta y eficiente para análisis de datos. Permite distribuir los datos a varios nodos, permitiendo paralelizar la fase de mapeo. Nuestros expertos usan esta tecnología desde su salida a producción.

  • Client: Redshift
  • Category: Almacenamiento en columnas en paralelo
